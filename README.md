# 🪑 Kajal Furniture – Business Website

A modern, responsive business website built for **Kajal Furniture**, a local furniture shop in Rourkela, Odisha.
The website showcases premium wooden furniture products and provides customers with an easy way to explore furniture categories and contact the shop.

---

## 🌐 Live Website

Deployed using **Vercel**

## 🚀 Getting Started

Clone the repository:

```
git clone https://github.com/kprem20/kajalfurniturewebsite.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Build for production:

```
npm run build
```

---

## 🚀 Deployment

The website is deployed using **Vercel**.

Deployment settings:

* Framework: Vite
* Build Command: `npm run build`
* Output Directory: `dist`
